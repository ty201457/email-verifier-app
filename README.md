# Email Verifier Web App

This is a Flask-based web application that allows you to validate the existence of email addresses using SMTP.

## Features
- Single email address validation via form
- Batch validation via CSV file upload (max 100MB)
- Deployable on Render with no local Python installation required

## Live Demo
Once deployed to Render, your app will be accessible at:  
`https://your-app-name.onrender.com/`

## How to Deploy (on Render)
1. Upload this project to GitHub
2. Create a new Web Service on [Render](https://render.com)
3. Link your GitHub repo and Render will auto-deploy
